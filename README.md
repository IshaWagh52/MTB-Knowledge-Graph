# MTB-KG

MTB-KG is a knowledge graph framework for studying *Mycobacterium tuberculosis* (MTB) genes and their biological relationships. The project combines different types of biological data into one unified graph, including protein interactions (STRING), transcriptional regulatory networks (TRN), literature-derived associations, MycoBrowser annotations, GO similarity, MORPH pathway predictions, and Graph Neural Network (GNN) virulence predictions.

The aim of MTB-KG is to help prioritize important MTB genes, explore how genes are connected, and better understand MTB biology using a systems-level approach. The framework also keeps track of evidence sources and distinguishes between regulatory relationships and association-based relationships.

The repository includes tools for knowledge graph construction, literature mining, MORPH and GNN integration, evidence-based gene ranking, and interactive graph exploration.
